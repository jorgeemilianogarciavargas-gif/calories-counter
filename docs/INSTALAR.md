# Instalar en el movil

URL esperada cuando GitHub Pages este activo:

```text
https://jorgeemilianogarciavargas-gif.github.io/calories-counter/
```

## Android

1. Abre la URL en Chrome.
2. Toca el menu de tres puntos.
3. Toca `Agregar a pantalla principal` o `Instalar app`.
4. Abre `Macros` desde tu pantalla de inicio.

## iPhone

1. Abre la URL en Safari.
2. Toca el boton de compartir.
3. Toca `Agregar a pantalla de inicio`.

La app guarda tus datos en el almacenamiento del navegador del telefono.
